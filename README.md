# Insurance Mini App - Alipay

تطبيق Mini App لإصدار وثيقة التأمين عبر Alipay.

## الميزات

- تسجيل الدخول عبر Alipay Mini App API
- إنشاء عرض سعر للتأمين
- إدارة المستفيدين
- معالجة الدفع
- عرض نتيجة إصدار الوثيقة

## التقنيات المستخدمة

- React 19
- TypeScript
- Vite
- React Router
- Zustand (State Management)
- React Hook Form + Zod (Form Validation)
- Tailwind CSS
- shadcn/ui
- Lucide React (Icons)
- Axios (HTTP Client)
- TanStack Query (React Query)

## التثبيت

```bash
cd my-react-app
npm install
```

## التشغيل

```bash
cd my-react-app
npm run dev
```

## البناء للإنتاج

```bash
cd my-react-app
npm run build
```
